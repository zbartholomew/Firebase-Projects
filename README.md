# Firebase Projects

## Firebase Chat App
  - This chat app follows the curriculum of udacity's Firebase in a Weekend course.
  - The goal of the app is to store messages and photos within Firebase and show the appropriate objects within the chat.
  - The app demonstrates Firebase's:
    - Realtime database
    - Storage (photos)
    - Authentication (Firebase UI)
    - Analytics
    - Notifications
    - Remote Configurations (message lengths)
